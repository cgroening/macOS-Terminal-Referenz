# Beliebte Programme für die Kommandozeile

| **Programm**                                               | **Beschreibung**                                                                                                      |
| ---------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------- |
| [bat](https://github.com/sharkdp/bat)                      | Wie `cat`, aber mit zusätzlichen Funktionen wie Syntaxhighlighting und Git-Integration                                |
| [bc](https://www.gnu.org/software/bc/)                     | Kommandozeilen-Taschenrechner für mathematische Ausdrücke                                                             |
| [claude-code](https://github.com/anthropics/claude-code)   | Nutzung von Claude-AI-Modellen, besonders für Programmier- und Analyse-Workflows                                      |
| [fzf](https://github.com/junegunn/fzf)                     | Fuzzy-Finder für die Kommandozeile – interaktives Suchen in Dateien, Verzeichnissen, Git-History usw.                 |
| [gitui](https://github.com/extrawurst/gitui)               | Terminal-UI für Git – intuitive, schnelle grafische Bedienung von Git-Workflows direkt im Terminal                    |
| [helix](https://github.com/helix-editor/helix)             | Moderne modale Texteditor-Alternative zu Vim/NeoVim mit LSP-Integration und innovativem Bedienkonzept                 |
| [htop](https://github.com/htop-dev/htop)                   | Interaktiver Prozess- und Ressourcenmonitor, erweiterte Version von `top`                                             |
| [mc](https://github.com/MidnightCommander/mc)              | Midnight Commander: Dateimanager im Terminal in Anlehnung an Norton Commander (Total Commander für die Kommandozeile) |
| [lsd](https://github.com/lsd-rs/lsd)                       | Wie `ls`, aber mit Farben, Icons, Baumansicht, besserem Format und erweiterten Optionen                               |
| [micro](https://github.com/zyedidia/micro)                 | Benutzerfreundlicher Terminal-Texteditor mit Kurzbefehlen wie STRG+C, STRG+V, usw.; Alternative zu `nano`             |
| [neovim](https://github.com/neovim/neovim)                 | Erweiterter, moderner Vim-Fork mit Plugin-System, LSP und besserer Integration in moderne Entwicklungs-Workflows      |
| [nnn](https://github.com/jarun/nnn)                        | Extrem schneller, minimalistischer Dateimanager mit Plugin-System und vielen Integrationen                            |
| [pandoc](https://github.com/jgm/pandoc)                    | Universeller Dokumenten-Konverter – wandelt praktisch jedes Format in jedes andere um                                 |
| [rsync](https://github.com/RsyncProject/rsync)             | Leistungsstarkes Tool für Dateisynchronisation und Backups, unterstützt Delta-Übertragung und SSH                     |
| [tldr](https://github.com/tldr-pages/tldr)                 | Vereinfachte, praxisorientierte Hilfeseiten mit Beispielen – schnelle Alternative zu `man`-Pages                      |
| [tree](https://sourabhbajaj.com/mac-setup/iTerm/tree.html) | Zeigt Verzeichnisstrukturen im Baumformat an – übersichtliche Alternative zu rekursivem `ls`.                         |
